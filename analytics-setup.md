# 방문자 통계 추적 설정 가이드

## 🎯 Google Analytics 설정 (권장)

### 1. Google Analytics 계정 생성
1. [Google Analytics](https://analytics.google.com) 접속
2. "시작하기" → "계정 만들기"
3. 계정 이름: "마음 한첩 웹사이트"
4. 속성 이름: "마음 한첩"
5. 플랫폼: 웹
6. 웹사이트 URL: `https://your-domain.com` (또는 임시 URL)
7. 스트림 이름: "마음 한첩 메인"

### 2. 추적 ID 확인
- 생성 후 "데이터 스트림" → "웹 스트림" 선택
- "측정 ID" 복사 (형식: G-XXXXXXXXXX)

### 3. 코드 교체
`templates/cinema.html` 파일에서:
```javascript
gtag('config', 'GA_MEASUREMENT_ID');
```
를 실제 ID로 변경:
```javascript
gtag('config', 'G-XXXXXXXXXX');
```

### 4. 통계 확인
- Google Analytics 대시보드에서 실시간 방문자 수 확인
- "보고서" → "실시간" → "개요"에서 접속자 수/페이지뷰 확인
- "획득" → "트래픽 획득"에서 접속 경로 분석

---

## 📊 확인 가능한 통계

### 기본 지표
- **총 방문자 수**: 누적 방문자
- **실시간 방문자**: 현재 접속 중인 사람 수
- **페이지뷰**: 페이지 조회 수
- **세션**: 방문 세션 수

### 접속 경로 분석
- **소스/매체**: 어디서 왔는지 (직접 입력, 검색, 소셜 등)
- **랜딩 페이지**: 처음 방문한 페이지
- **사용자 흐름**: 사이트 내 이동 경로

### 기기/지역 정보
- **국가/도시**: 방문자 지역
- **기기**: 모바일/데스크톱 비율
- **브라우저/OS**: 사용 환경

---

## 🔧 대안: 간단 방문자 카운터

Google Analytics가 어려우면 간단한 카운터를 추가할 수 있습니다.

### 설정 방법
1. 무료 카운터 서비스 가입 (예: StatCounter, Flag Counter)
2. 코드 복사해서 HTML에 삽입
3. 대시보드에서 통계 확인

---

## 💡 팁

- **테스트**: 설정 후 자신의 브라우저에서 접속해서 통계가 올라가는지 확인
- **프라이버시**: Google Analytics는 GDPR 준수 (쿠키 배너 필요시 추가)
- **보고**: 심사위원들에게 "Google Analytics로 접속 통계 확인 가능"이라고 언급 가능

## 🚨 주의사항

- **GA_MEASUREMENT_ID**를 실제 Google Analytics ID로 교체해야 작동합니다
- 웹사이트를 호스팅해야 통계가 쌓입니다 (로컬 파일로는 안 됨)
- 심사위원 접속 시점부터 통계가 쌓입니다
